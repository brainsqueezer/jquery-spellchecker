MarkItUp Example

Overview

This page shows how this spellchecker plugin can be used to check the spelling of a MarkItUp editor, using google's spell checking service. The plugin also comes with the option of using pspell. If you are able to use pspell, you will have greater control over your dictionary.

Click on the "Check spelling" button on the editor toolbar to see the plugin in action.

If you're looking for a bare-bones demo then take a look at the simple demo page.

Please read the documentation on how to setup the spellchecker first, and then do the following to get the spellchecker integrated into markitup.

Add the following object to your markupSet in the global "mySettings" object, in your sets.js file 

{
	name: 'Spellcheck', 
	className: 'spellcheck', 
	beforeInsert: function(markitup) { 
		miuSpellchecker(markitup); 
	} 
}

Stick this function within your sets.js file: 

function miuSpellchecker(markitup){
        if ($(markitup.textarea).hasClass("spellcheck-editor")) {
                $(markitup.textarea).spellchecker("remove");
        } else {
                $(markitup.textarea)
                .spellchecker({
                        engine: "pspell",
                        rpc: markitup.root+"utils/spellchecker/checkspelling.php",
                        after: ".markItUpFooter"
                })
                .spellchecker("check", function(result){
                        if (result) {
				$(markitup.textarea).spellchecker("remove");
                                alert('There are no incorrectly spelt words.');
                        }
                });
        }
}

Add the following CSS to your sets style.css file: 

.markItUp .spellcheck a {
        background-image:url(images/spellcheck.png);
}
  • Create the directory 'markitup/utils/spellchecker', and place checkspelling.php in there.
  • Place spellcheck.png in your set 'images' folder.
