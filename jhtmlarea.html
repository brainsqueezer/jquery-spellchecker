jHtmlArea Demo

Overview

This page shows how this spellchecker plugin can be used to check the spelling of a WYMeditor editor, using google's spell checking service. The plugin also comes with the option of using pspell. If you are able to use pspell, you will have greater control over your dictionary.

Click on the "Check spelling" button on the editor toolbar to see the plugin in action.

If you're looking for a bare-bones demo then take a look at the simple demo page. You can find a full featured demo here.

Please read the documentation on how to setup the spellchecker first, and then do the following to get the spellchecker integrated into the WYMeditor.

Download the neccesary spellchecker files, and check the image paths are correct in the spellchecker.css file.

Download the WYMeditor source files.

Include the following in the head section of your html document: 

<link rel="stylesheet" type="text/css" media="screen" href="css/spellchecker.css" />
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.spellchecker.js"></script>
<script type="text/javascript" src="wymeditor/jquery.wymeditor.min.js"></script>

Use the following to instantiate WYMeditor, and add the spellchecker button to the toolbar:

$('#wymeditor').wymeditor({

	postInit: function(wym){

		//construct the button's html
		var html = "<li class='wym_tools_newbutton'>"
		+ "<a name='NewButton' href='#'"
		+ " style='background-image:"
		+ " url(/img/spellcheck_icon.png)'>"
		+ "Do something"
		+ "</a></li>";

		//add the button to the tools box
		$(wym._box)
		.find(wym._options.toolsSelector + wym._options.toolsListSelector)
		.append(html);

		//handle click event
		$(wym._box)
		.find('li.wym_tools_newbutton a').click(function() {

			// initiate the spellchecker
			$(wym._doc.body)
			.spellchecker({
				url: "checkspelling.php",
				lang: "en", 
				engine: "pspell",
				suggestBoxPosition: "above",
				innerDocument: false,
				wordlist: {
					action: "append",
					element: $(".wym_area_main")
				},  
			})
			.spellchecker("check");
		});
	}
});

That's it!
