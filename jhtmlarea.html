jHtmlArea Demo

Overview

This page shows how this spellchecker plugin can be used to check the spelling of a jHtmlArea editor.

Click on the "Check spelling" button on the editor toolbar to see the plugin in action.

Please read the documentation on how to setup the spellchecker first, and then do the following to get the spellchecker integrated into the jHtmlArea editor.

Download the neccesary spellchecker files, and check the image paths are correct in the spellchecker.css file.

Download the jHtmlArea source files.

Include the following in the head section of your html document: 

<link rel="stylesheet" type="text/css" media="screen" href="css/spellchecker.css" />
<link rel="stylesheet" type="text/css" media="screen" href="jhtmlarea/style/jHtmlArea.css" />
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.spellchecker.js"></script>
<script type="text/javascript" src="jhtmlarea/scripts/jHtmlArea-0.7.0.js"></script>

Use the following to instantiate jHtmlArea, and add the spellchecker button to the toolbar:

$("#jhtmlarea").htmlarea({
	toolbar: [
		["bold", "italic", "underline"],
		["p", "h1", "h2", "h3", "h4", "h5", "h6"],
		["link", "unlink", "|", "image"],			 
		// custom spellcheck button
		[{
			css: "check-spelling-button",
			text: "Check spelling",
			action: function(btn) {
				// initiate the spellchecker
				$(this.editor.body)
				.spellchecker({
					url: "checkspelling.php",
					lang: "en", 
					engine: "pspell",
					suggestBoxPosition: "above",
					innerDocument: false,
					wordlist: {
						action: "after",
						element: $(".jHtmlArea")
					},  
				})
				.spellchecker("check", function(result){
					(result) && alert('There are no incorrectly spelt words.');
				});
			}
		}]
	]
});
